# How to Install ConuPlanner VSB Exporter (Developer Mode)

Since this extension is not yet in the public Chrome Web Store, you need to install it manually. It takes 30 seconds.

### Step 1: Unzip
1.  Unzip this folder (`vsb-extension`) to a known location (e.g., your Desktop or Downloads).

### Step 2: Open Chrome Extensions
1.  Open Google Chrome.
2.  In the URL bar, type: `chrome://extensions` and press Enter.
3.  **Important:** In the top-right corner, toggle the **"Developer mode"** switch to **ON**.

### Step 3: Load the Extension
1.  Click the **"Load unpacked"** button (top-left).
2.  Select the **unzipped folder** (`vsb-extension`) you created in Step 1.
    *   *Note: Select the folder itself, not a file inside it.*

### Step 4: Test It
1.  Go to [vsb.concordia.ca](https://vsb.concordia.ca).
2.  Create a schedule as usual.
3.  You should see the **"Export Schedule"** button pop down from the top of the screen!
